# Book API

## Description

This is an API for managing books.

## Technologies

- NestJS
- TypeORM
- PostgreSQL
- Redis

## Running the app

```bash
docker compose up
```

## Running the tests

### Unit tests

```bash
docker compose exec -it app npm run test
```

### E2E tests

```bash
docker compose exec -it app npm run test:e2e
```
